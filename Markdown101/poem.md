## This is a message about what a .md file is:
A .md file or markdown fie is a plain text file formated in a way to display syntax is a simple and readable style that can be also called and displayed by other programs and/or websites. .md files are pure text so while they can contain commands such as print("x=10") it does not actaully mean anything in the .md file as there is no command processing. It is possible to write code in a markdown file but would need to be uploaded or readed by another program and changed from a .md file to another format. Works well when wanting to display just simple text and can act as readme files that can be accessed and formated. Also side not does not have any extra features so any miss spelling or grammatical errors in this text is left to show that point.

# The Raven

Once upon a modnight dreary, while I pondered, weak and weary,
Shadows clung to the wooden needles, lean and narrow,
Through the wooden doors, creeped and slithered,
Sights upon this land do ponder.

Dance upon this forign scape, nymphs of desire,
Lead these rivers through unknown yearns,
Clash and ceed, upon the curtain show,
Blaze the light, to chase the flicker.

See through the veil as the beyond beckons,
Beware the sleeping day for what follows,
Run from the game unknown.
Settle upon the storm bed.

Return to rest, lest thoughts wander,
Receed upon these shallow shores,
softly lay on the bed of thorns,
Do not distrub the waking night.
